# Financial Data Automation & ETL Pipeline

## Project Description
An automated data pipeline built in Python and SQL to streamline multi-currency (USD/CAD) transaction verification and eliminate manual data processing bottlenecks.

## Features
- **Data Cleaning (ETL):** Uses Python (Pandas) to ingest and standardize unstructured transaction logs.
- **Currency Standardization:** Automatically converts USD transactions to CAD using live exchange logic.
- **SQL Analytics:** Ingests cleaned datasets into a SQLite database to run complex joins and isolate high-value or pending variances instantly.

## Tech Stack
- **Language:** Python, SQL
- **Libraries:** Pandas, SQLite3, IO
